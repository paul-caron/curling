{\footnotesize\ttfamily enum class \mbox{\hyperlink{classcurling_1_1Request_a7176a85fe8cab5746fd03a57f0e7cb5f}{curling\+::\+Request\+::\+Http\+Version}}\hspace{0.3cm}{\ttfamily [strong]}}



Specifies the HTTP version to be used for the request. 

Allows explicit control over the HTTP version that libcurl should use when performing a request. If the selected version is not supported by the libcurl build or the server, fallback behavior may occur.

\begin{DoxyNote}{Note}
If you request a version not supported by libcurl at runtime, Curling will throw a \doxylink{classcurling_1_1LogicException}{Logic\+Exception} during configuration. If the server doesn\textquotesingle{}t support the version (e.\+g., HTTP/2), libcurl may fall back to an older version without error. 
\end{DoxyNote}
